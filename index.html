Projet #1 - AIML Database ChatBot With UI

In my second year, I embarked on a project to develop a chatbot as part of my AIML subject coursework. The chatbot was designed to handle various user queries by leveraging Natural Language Processing (NLP) and machine learning techniques. The core functionalities included text preprocessing, intent recognition, and response generation. Key NLP techniques employed were tokenization, stemming, and named entity recognition, which facilitated the chatbot's understanding of user inputs. For machine learning, I utilized classification algorithms such as Naive Bayes and Support Vector Machines (SVM) to enhance the chatbot's ability to categorize user intents and generate appropriate responses. In my final year, I sought to enhance this project by integrating a more sophisticated user interface using Flask. This upgrade involved designing a dynamic web interface that improved the overall user experience and interaction with the chatbot The Flask application allowed for real-time communication with the chatbot, incorporating features like user authentication, session management, and responsive design to ensure a seamless and engaging experience. The interface was visually refined with enhanced colors and layout adjustments, aligning with modern design principles to provide a more intuitive and user-friendly interaction. Technologies Utilized : Flask, Python, NLTK techniques, HTML and Machine Learning
